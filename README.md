# PROJECT SYNOPSIS

This project addresses a real-time challenge faced by the HR department at Atliq Technologies using Power BI.

## Problem Statement
The HR team at Atliq Technologies provided a spreadsheet containing employee attendance data for the first three months of FY 2022–23. The dataset includes attendance dates and corresponding key codes, which are typically used to respond to HR-related queries. The challenge lies in transforming this raw data into meaningful insights.

## Objective
The objective of this project is to build a comprehensive Power BI dashboard that aligns with stakeholder requirements and delivers actionable insights. The dashboard focuses on:
- Understanding employee working preferences
- Identifying patterns and root causes in attendance behavior
- Supporting capacity planning decisions

The solution incorporates continuous stakeholder feedback to ensure iterative improvement and relevance.

---

## Unlocking Valuable Insights

### Absence Management
- Identifies trends in sick leave and absenteeism
- Enables HR to investigate root causes and implement preventive measures (e.g., sanitization, vaccinations)

### Optimal Resource Utilization
- Helps optimize office space and infrastructure costs in a hybrid work environment
- Supports better capacity planning through attendance trend analysis

### Strategic Event Planning
- Identifies optimal days for team activities based on employee presence
- Maximizes participation in events such as team lunches or gatherings

### Performance Analysis
- Examines correlations between attendance and productivity
- Supports identification of performance-related attendance patterns

### Employee Satisfaction & Work-Life Balance
- Highlights employee preferences for remote work or flexible schedules
- Helps HR design policies to improve well-being and satisfaction

### Trend Identification
- Detects long-term and seasonal attendance patterns
- Supports workforce planning for peak and off-peak periods

### Compliance Monitoring
- Tracks adherence to company attendance policies
- Flags excessive absenteeism and irregular attendance behaviors

### Data-Driven Decision Making
- Empowers HR with actionable insights through visualization
- Enhances decision-making in workforce management and engagement strategies

---

## Key Takeaways

- Utilized Power Query for efficient data extraction, cleaning, and transformation
- Addressed real-world data challenges such as missing values and inconsistencies
- Leveraged DAX (Data Analysis Expressions) to build advanced metrics and KPIs
- Collaborated with stakeholders to refine requirements and identify valuable insights
- Applied data visualization best practices to highlight key insights effectively
- Developed insight-driven reports that encouraged deeper analytical thinking
- Delivered a Minimum Viable Product (MVP) and iteratively improved it based on feedback
- Built solutions aligned with dynamic HR and business requirements
